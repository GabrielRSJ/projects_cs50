# 🦆 WEEK 0 - CS50: Introdução à Lógica com Scratch

## 📝 O que aprendi

Na primeira semana do CS50, fui apresentada aos fundamentos da lógica de programação utilizando o **Scratch**, uma linguagem visual baseada em blocos.

Mesmo sendo uma ferramenta simples, o Scratch me ajudou a entender conceitos que são a base de TODAS as linguagens de programação.

## 🧠 Conceitos que pratiquei

| Conceito | O que significa | Como usei no Scratch |
|----------|-----------------|----------------------|
| **Estrutura de um código** | Todo programa tem uma sequência lógica | Organizei blocos em ordem para criar animações |
| **Sistemas de repetição (loops)** | Repetir ações sem precisar copiar blocos várias vezes | Usei blocos `repetir` e `sempre` para fazer personagens se moverem continuamente |
| **Expressões booleanas** | Condições que resultam em verdadeiro ou falso | Blocos `se` e `se/senão` para decisões |
| **Variáveis** | Guardar informações que podem mudar | Criei variáveis como "pontuação" e "vidas" |


## 💡 Desafios que enfrentei

- **Entender a diferença entre repetir para sempre vs repetir x vezes** – No começo confundi, mas testando vi a diferença
- **Organizar os blocos sem bagunçar** – Aprendi a importância de manter o código limpo mesmo visualmente
- **Lógica de pontuação** – Como fazer aumentar só quando passar pelo obstáculo, não o tempo todo

## 🎯 Conclusão

Mesmo sendo uma ferramenta infantil, o Scratch me ensinou que **todo programador precisa pensar em:**
- Sequência
- Condições
- Repetições
- Dados (variáveis)

Essa base me ajudou em C e qualquer outra linguagem, a logica é a mesma o que muda é a sintaxe.
